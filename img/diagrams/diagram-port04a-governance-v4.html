Governance Model

The Loop

Every piece of portfolio content cycles through the same human–AI collaboration loop. Della sets intent; Claude generates; both evaluate; the loop continues until it ships.

Continuous Cycle
Human intent → AI craft → Human judgment
Della
Set Intent
Goals, constraints, voice rules, target company
Every task starts with explicit intent — what company this targets, what voice register to use, what the success criteria are.
Both
Brief
Design brief, skill invocation, codesign step 0
Structured briefs load pattern libraries, design references, and quality benchmarks before generation begins.
Claude
Generate
Content, code, diagrams, case study copy
HTML diagrams, case study narrative, resume variants, outreach emails — all generated from verified source material.
Claude
Self-Check
Voice lint, quality gate, render and verify
Automated checks: banned-patterns.yaml scan, verified-facts-registry check, PDF render for page counts, visual QA.
Both
Decision Gate
Della reviews. Approves → ship. Issues → redirect back through the loop.
Human judgment as the final gate — checking voice, accuracy, and whether the output actually solves the original intent.
Della
Redirect
Refined constraints, new direction, back to intent
Not failure — refinement. Each redirect narrows the solution space with better constraints from what was learned.
Ship
Passes all checks
Della leads
Claude leads
