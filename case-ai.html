Case Study 02 of 05 — Pioneered AI-forward design practice at Reddit

Keyword Landing Pages / AI

I designed Reddit's first LLM product — and the governance model, component library, and prompt workflow that became the foundation for Reddit Answers.

Senior Product Designer
SEO & Discovery
Jun 2023 – Dec 2023
Web (SEO surfaces)

Reddit wanted to consolidate fragmented SEO traffic using LLMs. Straightforward product idea — the organization had no precedent for AI-generated content. No identity system, no evaluation criteria, no framework for non-deterministic output. I led the design of a governance model that defined how LLMs would appear, sound, and behave across the platform — shipped iteratively over six months.

Visual Identification

Before touching UI, I partnered with OrangeRed (brand), legal, and design systems to define what AI-generated content would be at Reddit — not just how it would look, but what identity it carried and what legal implications followed.

Transparency was the design constraint. User research confirmed people valued knowing content was AI-generated, but this was peak AI backlash — the signal had to build trust without feeling like a trend.

Working with the design systems and brand teams, I explored the full spectrum of identification patterns — and tested them across surfaces well beyond the initial product.

Exploration range — identification patterns from heavily branded to near-invisible, tested across SEO landing pages, composer, comment threads, profiles

The component library, LLM identification patterns, and user feedback UI shipped as reusable system components — designed from the start to scale to wherever AI content appeared next.

Tone & Voice

The closer summaries stayed to original posts, the more users trusted them. Broad generalizations — a consistent failure mode in early outputs — killed trust immediately. Every summary needed direct attribution and links so users could go deeper instead of guessing from a search page.

I drove the team through the harder question: how to synthesize across communities without losing the source. The model’s range was narrow — too little context produced empty platitudes, too much produced verbatim quotes with zero synthesis.

Summary display explorations — length variants, source attribution placement (inline vs. post bank), expand/collapse patterns

The synthesis display patterns and truncation logic we landed on became reusable across later products.

Prompt Design Workflow

Previously, prompt iteration was one engineer, vibes-based. I introduced a structured evaluation matrix — multiple prompts tested simultaneously against live data from Reddit’s highest-traffic SEO verticals, scored on defined parameters: summary quality, source relevance, synthesis depth, transparency, community diversity, post engagement.

My core contribution was translating subjective design criteria into rubrics engineers could operate against — bridging “this feels wrong” and “this scores below threshold on source attribution.” The rubric evolved with each production cycle as we learned what benchmarks couldn’t predict.

Prompt evaluation matrix — parameters, scoring criteria, iteration cycle from vibes-based to structured

Post-processing stayed minimal. Heavy backend cleanup masked the nondeterministic behavior the system needed to handle honestly.

Failure States & User Control

Below-threshold quality triggered an empty state — not a bad summary. Users could flag responses, suggest improvements, or opt out entirely. These signals fed directly back into the quality loop.

Results

This project became the origin of Reddit Answers. The governance model, identity system, and prompt workflow I built were handed to a new team spun up to bring synthesis into search — where research had shown it belonged. Sending users from one search-like surface to another was the dead end that clarified the real opportunity.

Before/after — KLP surface vs. synthesis in search (Reddit Answers)

What scaled: the LLM identification component library, synthesis display patterns, user feedback UI, prompt evaluation rubric, and governance framework. Reddit Answers inherited these artifacts directly and shipped on top of them.
